# 安装 serviceloader 

```
cd $GOPATH/src/steve/serviceloader 
go install 
```

# 使用 plugin 方式编译服务

```
# 编译 room 服

cd $GOPATH/src/steve/room
go build -buildmode=plugin 
```

# 启动服务 

```
# 启动 room 服务

cd $GOPATH/src/steve/room
serviceloader room --config=config.yml 
```
